月明茶房

静寂と温もりを感じるデザイン

制作背景

「静寂な空間で、ゆっくりと時間を過ごす」というカフェのコンセプトを
Webサイトでも体験できるよう、落ち着いた雰囲気とインタラクティブな演出を重視しました。
訪問者が実際にカフェを訪れたかのような、心地よい体験を提供することを目指しました。

概要

「月明かりの下、時を忘れる静謐な喫茶。」をテーマに、
コンセプト設計からコーディングまでを担当しました。
暖かい色味と陰影を感じるデザインを意識しました。

  • 役割：UI/UXデザイナー / フロントエンド開発者
  • 制作期間：3週間（企画1週間 / デザイン1週間 / 実装1週間）
  • 使用ツール：Figma / Photoshop / Illustrator / HTML / CSS / JavaScript / Three.js
  • 担当範囲：
    • コンセプト設計・ペルソナ設定
    • ワイヤーフレーム・デザインカンプ作成
    • HTML/CSS/JavaScriptでのコーディング
    • Three.jsを使った3D実装

ワイヤーフレーム

デザインカンプ

デザインコンセプト

月明かりを想起させる柔らかいトーンと、黒を基調とした静寂感を融合。
カフェの時間がゆっくり流れるように、スクロールやホバーの動きも控えめにデザインしました。

デザインのポイント

  • 配色：月明かりを連想させる温かみのあるベージュと、夜の静けさを表現する黒を基調
  • タイポグラフィ：和風の雰囲気を演出する明朝体を使用
  • アニメーション：ゆったりとした時間の流れを表現する控えめな動き

技術実装

ローディングアニメーションとThree.jsを使用したスクロール連動式3Dオブジェクトを取り入れました。
ユーザー体験を損なわないよう、パフォーマンスとビジュアルのバランスを重視しました。
