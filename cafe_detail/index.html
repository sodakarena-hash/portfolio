月明茶房

静寂と温もりを感じるデザイン

概要

「月明かりの下、時を忘れる静謐な喫茶。」をテーマに、
コンセプト設計からコーディングまでを担当しました。
暖かい色味と陰影を感じるデザインを意識しました。

  • 使用ツール： Figma / Photoshop / Illustrator / HTML / CSS / JavaScript
